정규식

앵커문자

문자열의 앞과 뒤를 구부해주는 정규표현식 기호

메타문자

자바 스크립트 정규 표현식 객체가 갖는 가장 유용한 기능

메타문자를 이용한 주민번호 확인

주민번호 입력 :

추가 메타 문자

\d : 숫자
\w : 아무 단어 (숫자 포함)
\s : 공백 문자 (탭, 띄어쓰기, 줄바꿈)
\D : 숫자 아님
\W : 아무 단어 아님
\S : 공백 문자 아님
() : 그룹화 ?= : 결가에 포함하지 않고 기본 표현식 뒤의 그룹과 일치(그룹의 패턴을 차례대로 일치하는지 검사) . : 줄바꿈을 제외한 모든 문자 * : 0개 이상의 이전 토큰과 일치

수량 문자

a+ : a가 적어도 1개 이상
a* : a가 0개 또는 여러개
a? : a가 0개 또는 한개
a{5} : a가 5개
a{2,5} : a가 2개, 3개, 4개 다 찾겠다
a{2, } : a가 2개두개 이상
a{ ,2} : a가 2개두개 이하
